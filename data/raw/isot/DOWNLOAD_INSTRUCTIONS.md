# ISOT Fake News Dataset

## Option 1: Kaggle CLI
```bash
pip install kaggle
kaggle datasets download -d clmentbisaillon/fake-and-real-news-dataset -p data/raw/isot --unzip
```

## Option 2: Manual Download
1. Go to: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset
2. Download and extract the zip
3. Place `True.csv` and `Fake.csv` in this directory
